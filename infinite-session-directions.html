1. Setup your facebook application as described in the README.txt for this plugin - you must establish the facebook application first, to get an "api key" and a "secret"
  2. Fill out the rest of the configuration on the WPBook configuration screen and submit
  3. Log out of Facebook - don't just close open browser windows but use the log out button inside Facebook
  4. Visit the following page, which will ask you to login, and should show you your infinite session key: [blog url]/wp-content/themes/wp-facebook/infinitesession.php
    (Where blog url is replaced with the url to your blog
  5. That url will redirect you to a facebook login - log in with your facebook credentials and you should see a screen which tells you your infinite session key.
  6. Enter that key in the WPBook configuration screen
  7. Once you have an infinite session key, visit the Edit Applications page in Facebook.
  8. Click on "Edit Settings" for your blog application
  9. Click on the profile tab in the resulting popup
  10. Add the profile box
By default, the profile box is added to your "boxes" tab if you have more than 5 applications installed - but you can move it back to your main profile using the little edit icon in the upper right hand corder of the profile box, in your boxes page.
